# Modbus Data Fromat

## show data

```
A1 00 00 FF 00 00 0B AB [data] 0c
```
Length: 8 + sizeof(DATA) + 1
|标志|位数|说明|
|:---|:---:|:---|
|A1 00	|2|地址码|
|00 FF	|2|两位显示设备数值 不实现功能暂时保留|
|00		|1|显示方向 0x00横向，0x01纵向，0x02横向垂直反转，0x03纵向水平反转|
|00		|1| 空位 |
|0B		|1|数据位数 从数据起始位后一位开始，到结束|
|AB		|2|数据起始位 位固定不变|
|DATA	|x|数据|
|0C		|1| CRC校验位|
